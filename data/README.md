# Sample Datasets

This directory contains demonstration datasets used by the Enterprise Financial Intelligence Platform.

## Files

### sample_financials.csv

Contains company financial metrics.

Columns:

- Company
- Revenue
- Profit
- Growth

---

### sample_forecast_data.csv

Contains historical revenue data used for forecasting demonstrations.

Columns:

- Year
- Revenue

---

## Purpose

These datasets are included solely for educational, demonstration, and portfolio purposes.
